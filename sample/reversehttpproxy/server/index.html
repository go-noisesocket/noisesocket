Protocol specification draft

Go implementation

Following ports are provided:

  • 13242 Selects sub-messages using server priority
  • 13243 Selects sub-messages randomly
  • 13244 Selects first sub-message

The /status page is available on these ports

(This ascii art logo is provided only for the purpose of increasing page size for testing big packets. It is also served via NoiseSocket ports)
